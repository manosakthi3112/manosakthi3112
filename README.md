| Layer | Technology | Role |
|:---|:---|:---|
| 🐍 Backend | Python 3.11, Flask | REST API + business logic |
| 📊 Data | yfinance, Pandas, NumPy | Market data ingestion & transformation |
| ⚡ Cache | Redis | Sub-100ms response via smart caching |
| 🔄 Jobs | Celery + Redis Broker | Background async data refresh |
| 📉 Charts | Chart.js, Plotly | Interactive candlestick & volume charts |
| 🐳 Infra | Docker, Docker Compose | One-command deployment |
| 🗄️ DB | PostgreSQL | Historical data persistence |

**Highlights:**
- ⚡ Sub-100ms API responses via Redis caching layer
- 📊 Interactive candlestick charts with zoom, pan, tooltip overlays
- 🔄 Celery background jobs — auto-refresh without blocking main thread
- 📈 Technical Indicators: Moving Averages, Bollinger Bands, RSI
- 🐳 Fully Dockerized — `docker-compose up` and it's live
- 🔒 Rate-limit aware with exponential backoff
